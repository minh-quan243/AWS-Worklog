## Event Report: “AI Agents, Prompt Engineering, and AIoT on AWS”

### Event Objectives

- Understand how AI Agents extend the capabilities of Large Language Models (LLMs).  
- Grasp the role of prompt engineering in improving AI output quality.  
- Explore real-world AIoT architectures built on AWS.  
- Gather ideas and insights to apply to cloud-based projects.  

### Speakers

- **Banh Cam Vinh** – Topic: Building AI Agents with Strands  
- **Nguyen Tuan Thinh** – DevOps Engineer, First Cloud AI Journey  
  Topic: Prompt Optimization  
- **Aiden Dinh** – Operation Engineer, Katalon  
  Topic: AIoT Projects  

### Key Highlights

#### AI Agents – The Next Evolution of AI

- Traditional LLMs mainly handle single-step tasks.  
- AI Agents can execute multi-step workflows, utilize tools, and make decisions.  
- Well-suited for automation systems and complex workflows.  

#### Building AI Agents with Strands

- Core components include system prompt, tools, and knowledge base.  
- Processing flow: input → reasoning → tool invocation → output.  
- Simplifies the development of autonomous AI systems.  

#### Prompt Engineering

- Key elements: role, task, context, constraints, and format.  
- Design principles:
  - Be clear and specific  
  - Avoid ambiguity  
  - Structure content logically  
- Advanced techniques: Chain-of-Thought and Retrieval-Augmented Generation (RAG).  
- Well-designed prompts improve output quality and reduce operational costs.  

#### AIoT Architecture on AWS

- Case study: a smart locker management system.  
- Hardware components: Raspberry Pi, Arduino, RFID, sensors, and cameras.  
- AWS services used: IoT Core, Lambda, DynamoDB, and Rekognition.  
- Goal: enhance automation and operational efficiency.  

### Key Takeaways

#### AI Agents Trend

- AI is shifting toward agent-based architectures rather than standalone LLM usage.  
- Enables higher levels of automation and multi-step workflow execution.  

#### Importance of Prompt Engineering

- Prompts directly impact output accuracy and quality.  
- Effective prompt design reduces costs and improves system performance.  

#### AWS for AIoT

- AWS provides a comprehensive ecosystem for building AIoT solutions.  
- Supports scalability and real-world deployment with ease.  

***

### Event Experience

The event **“AI Agents, Prompt Engineering, and AIoT on AWS”** provided valuable practical knowledge and a clearer perspective on building modern AI systems.

The session on **AI Agents** helped distinguish between traditional LLMs and autonomous agent-based systems. This introduced a new approach to designing complex automation workflows.

Additionally, the **Prompt Engineering** segment highlighted that output quality depends not only on the model but also heavily on how prompts are structured. Applying techniques like Chain-of-Thought and RAG demonstrated significant improvements.

The **AIoT** session offered an end-to-end system perspective, from hardware devices to cloud integration. It clarified how to connect physical components with AWS services to build complete solutions.

Overall, the event reinforced the idea that AI is not just about models, but about designing an integrated system where prompts, architecture, and integration all play critical roles.

***

## Conclusion

The event strengthened my understanding of AI Agents, prompt engineering, and AIoT architecture. It also provided practical ideas that can be directly applied to real-world cloud and system development projects.
