## Event Report: “AI Agents, Prompt Engineering, and AIoT on AWS”

### Event Objectives

- Understand how AI Agents extend the capabilities of large language models (LLMs).
- Master the role of prompt engineering in improving AI output quality.
- Explore real-world AIoT architectures deployed on AWS.
- Gather ideas and practical experience to apply to cloud projects.

### Speakers

- **Banh Cam Vinh** – Topic: Building AI Agents with Strands  
- **Nguyen Tuan Thinh** – DevOps Engineer, First Cloud AI Journey  
  Topic: Prompt Optimization  
- **Aiden Dinh** – Operation Engineer, Katalon  
  Topic: AIoT Projects  

### Key Highlights

#### AI Agents – The Next Direction of AI

- Traditional LLMs mainly handle individual tasks.
- AI Agents can execute multi-step actions, use tools, and make decisions.
- Suitable for automation systems and complex workflows.

#### Building AI Agents with Strands

- Core components include: system prompt, tools, and knowledge base.
- Workflow: input → reasoning → tool invocation → output.
- Simplifies the development of autonomous AI systems.

#### Prompt Engineering

- Key elements: role, task, context, constraints, and format.
- Prompt design principles:
  - Clear and specific  
  - Avoid ambiguity  
  - Structure content properly  
- Advanced techniques: Chain-of-Thought, Retrieval-Augmented Generation (RAG).
- Well-designed prompts improve output quality and optimize costs.

#### AIoT Architecture on AWS

- Case study: smart locker management system.
- Technologies used: Raspberry Pi, Arduino, RFID, sensors, and cameras.
- Integrated with AWS services such as IoT Core, Lambda, DynamoDB, and Rekognition.
- Aimed at automating processes and improving operational efficiency.

### Key Takeaways

#### AI Agent Trends

- AI is shifting toward agent-based models rather than standalone LLMs.
- Enhances automation and multi-step workflow processing.

#### Importance of Prompt Engineering

- Prompts directly impact output accuracy and quality.
- Good design reduces cost and improves system performance.

#### AWS Capabilities in AIoT

- AWS provides a comprehensive ecosystem for building AIoT systems.
- Easily scalable and practical for real-world deployment.

***

### Event Experience

The event **“AI Agents, Prompt Engineering, and AIoT on AWS”** provided practical knowledge and a clearer perspective on building modern AI systems.

The **AI Agents** session helped me clearly distinguish between traditional LLM models and autonomous agent-based systems. This opens up new approaches for building complex automation systems.

The **Prompt Engineering** section emphasized that output quality depends not only on the model but also heavily on prompt design. Applying techniques like Chain-of-Thought and RAG significantly improves results.

The **AIoT** segment offered an end-to-end system perspective—from hardware devices to cloud integration—helping me better understand how to build complete solutions using AWS services.

Overall, the event helped me realize that AI is not just about models, but a comprehensive system where prompt design, architecture, and integration all play critical roles.

***

## Conclusion

The event reinforced my understanding of AI Agents, prompt engineering, and AIoT architecture, while also providing valuable ideas that can be directly applied to real-world cloud projects.
