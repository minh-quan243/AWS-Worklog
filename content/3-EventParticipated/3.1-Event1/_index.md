# Summary Report: “AWS RE:INVENT 2025 RECAP (VIETNAM)”

### Event Details
- **Date & Time:** January 27, 2026  
- **Location:** AWS Vietnam Office (Floors 26 & 36), Ho Chi Minh City  
- **Role:** Attendee (Cloud Intern at FCJ)  

### Event Objectives

- Stay updated on the most important announcements from AWS re:Invent 2025 (Las Vegas).  
- Gain deeper insights into Generative AI, particularly Agentic AI and Amazon Bedrock.  
- Explore new optimizations in data storage and infrastructure (SageMaker, S3).  
- Connect with AWS Solution Architects and the local tech community.  

### Speakers

- **Mr. Thi** – Solution Architect (Topic: Generative AI & Agents)  
- **Mr. Tung** – Speaker (Topic: OpenSearch & Agentic Search)  
- **AWS Solution Architects & Account Management Team**  

### Key Highlights

#### Session 1: Generative AI & Agents
- **Amazon Nova Models**: Introduction of new high-performance foundation models.  
- **Bedrock Agents**: In-depth exploration of orchestration, workflows (flows), and new features such as **Memory** and **Guardrails**.  
- **Agentic AI**: Transition from simple chatbots to autonomous agents capable of executing multi-step workflows.  

#### Session 2: SageMaker Unified Studio & S3 Updates
- **Unified Studio**: A unified development environment (IDE) connecting Data Engineers, Data Scientists, and AI Engineers.  
- **S3 Tables**: Native support for Apache Iceberg table format directly in S3.  
- **S3 Vector**: A new feature enabling direct vector storage on S3, significantly reducing costs.  

#### Sessions 3 & 4: Search & Multimodal AI
- **OpenSearch Serverless**: Integration with Model Context Protocol (MCP) and Agentic Memory.  
- **Nova Multimodal Embeddings**: Transforming video and image data into vectors for search applications.  
- **Bedrock Data Automation**: Automated extraction of deep insights from multimedia content.  

#### Session 5: AI Infrastructure
- **SageMaker HyperPod**: Advanced management for large-scale GPU clusters.  
- **SageMaker MLflow**: End-to-end lifecycle management for Machine Learning projects.  

### Key Takeaways

#### The Future is "Agentic"
- **Autonomous workflows**: A shift from prompt engineering to **agent engineering**, where agents with memory can maintain context and perform complex tasks independently.  
- **Guardrails are essential**: As agents become more autonomous, strict policies and safeguards are critical.  

#### Data & Compute Optimization
- **Cost efficiency**: S3 Vector is a game-changer for vector-based applications (e.g., RAG) with limited budgets.  
- **Collaboration**: SageMaker Unified Studio streamlines workflows between data preparation and model training.  

### Application to Work

- **Project Integration (Security Platform):**
  - Evaluate **Bedrock Agents** to automate vulnerability scanning workflows (e.g., an agent that scans, analyzes logs, and drafts reports).  
  - Implement **S3 Vector** for efficient storage of logs and vulnerability signatures in the backend.  

- **Architecture Improvement**: Consider using **Cognito** for user management based on discussions with AWS Solution Architects.  

- **Best Practices**: Apply the “Serverless-first” mindset learned from networking sessions.  

### Event Experience

Attending **“AWS re:Invent 2025 Recap”** at the AWS Vietnam office marked an important milestone during my internship:

#### Broadening Perspectives
- The sessions highlighted that **Agentic AI** represents the near future. Demonstrations of Flow Agents inspired ideas for reporting features in our project.  
- Learning about **Multimodal RAG** expanded my understanding beyond text processing, especially for future use cases involving screenshot analysis in the Security Platform.  

#### Technical Validation
- The introduction of **S3 Vector** confirmed our team’s need for a cost-effective solution for log analysis.  
- Interacting with **Mr. Thi** and other Solution Architects helped clarify challenges related to **IAM policies** and **serverless best practices** encountered during Week 3.  

#### Community & Networking
- The AWS office atmosphere was dynamic, with developers sharing real-world challenges and experiences.  
- I had the opportunity to discuss our team project **“TheBois”** with industry experts and received valuable feedback on our proposed architecture.  

> In summary, this event not only updated my technical knowledge but also provided practical tools (Agents, S3 Vector) that can be immediately applied to the **Website Security Baseline Assessment Platform**.
