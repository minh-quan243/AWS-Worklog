# Final Report: “AWS RE:INVENT 2025 RECAP (VIETNAM)”

### Event Details
- **Date & Time:** January 27, 2026
- **Location:** AWS Vietnam Office (Floors 26 & 36), Ho Chi Minh City
- **Role:** Attendee (Cloud Intern at FCJ)

### Event Objectives

- Stay updated with the most important announcements from AWS re:Invent 2025 (Las Vegas).
- Deepen understanding of Generative AI, specifically Agentic AI and Amazon Bedrock.
- Explore new optimizations in data storage and infrastructure (SageMaker, S3).
- Connect with AWS Solution Architects and the local tech community.

### Speakers

- **Mr. Thi** – Solution Architect (Topic: Generative AI & Agents)
- **Mr. Tung** – Speaker (Topic: OpenSearch & Agentic Search)
- **AWS Solution Architects & Account Managers Team**

### Key Highlights

#### Session 1: Generative AI & Agents
- **Amazon Nova Models**: Introduction to new high-performance foundation models.
- **Bedrock Agents**: Deep dive into orchestration, workflows (flows), and new features such as **Memory** and **Guardrails**.
- **Agentic AI**: Transition from simple chatbots to autonomous agents capable of executing multi-step workflows.

#### Session 2: SageMaker Unified Studio & S3 Updates
- **Unified Studio**: A single development environment (IDE) connecting data engineers, data scientists, and AI engineers.
- **S3 Tables**: Native support for Apache Iceberg table format directly on S3.
- **S3 Vector**: A new feature that enables direct vector storage on S3, significantly reducing costs.

#### Session 3 & 4: Search & Multimodal AI
- **OpenSearch Serverless**: Integration with MCP (Model Context Protocol) and Agentic Memory.
- **Nova Multimodal Embeddings**: Converting images and videos into vectors for search.
- **Bedrock Data Automation**: Automatically extracting insights from multimodal content.

#### Session 5: AI Infrastructure
- **SageMaker HyperPod**: Advanced management for large-scale GPU clusters.
- **SageMaker MLflow**: End-to-end lifecycle management for machine learning projects.

### Key Takeaways

#### The Future is “Agentic”
- **Autonomous workflows**: A shift from “Prompt Engineering” to “Agent Engineering.” Agents with memory can maintain context and perform complex tasks without continuous human intervention.
- **Guardrails are essential**: As agents become more autonomous, strict policies and safety mechanisms are critical.

#### Data & Compute Optimization
- **Cost efficiency**: S3 Vector is a game-changer for projects requiring vector search (e.g., RAG) with limited budgets.
- **Collaboration**: SageMaker Unified Studio streamlines workflows between data preparation and model training.

### Application to Work

- **Project Integration (Security Platform):**
  - Evaluate **Bedrock Agents** to automate vulnerability scanning workflows (e.g., an agent that runs scans, analyzes logs, and drafts reports).
  - Implement **S3 Vector** to efficiently store logs and vulnerability signatures for the project backend.

- **Architecture Improvements:**
  - Consider adopting **Cognito** for user management based on discussions with Solution Architects.

- **Best Practices:**
  - Apply a **serverless-first** mindset learned from the sessions.

### Event Experience

Attending **“AWS re:Invent 2025 Recap”** at the AWS Vietnam office was a significant milestone in my internship:

#### Expanding Perspective
- The sessions clearly showed that **Agentic AI** is the near future. Seeing Flow Agent demos inspired ideas for reporting features in our project.
- Learning about **Multimodal RAG** highlighted capabilities beyond text processing—especially useful for future stages of the Security Platform involving screenshot analysis.

#### Technical Validation
- The introduction of **S3 Vector** validated our team’s need for a cost-effective solution for log analysis.
- Conversations with **Mr. Thi** and other Solution Architects clarified questions about **IAM policies** and **serverless best practices** that I encountered earlier.

#### Community & Networking
- The AWS office environment was dynamic, with developers sharing real-world challenges.
- I had the opportunity to discuss our team project with industry experts and receive valuable feedback on our proposed architecture.

> In summary, this event not only provided updated technical knowledge but also introduced concrete tools (Agents, S3 Vector) that I can immediately apply to the **Website Security Baseline Assessment Platform**.
