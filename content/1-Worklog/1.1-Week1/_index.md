### Week 1 Objectives:

* Get familiar with the internship environment and connect with members of the **First Cloud Journey (FCJ)** program.
* Understand the workflow, rules, and team culture.
* Create and configure an **AWS** account, successfully activate the **Free Tier**.
* Begin learning fundamental knowledge of **Cloud Computing**.
* Explore the main service groups of **AWS**.
* Get familiar with and practice using **IAM (Identity and Access Management)**.

### Week 1 Achievements:

* Gained an overview of the internship program and became familiar with the team working environment.

* Understood basic concepts of **Cloud Computing**, including:
  * Benefits of cloud computing
  * Service models (IaaS, PaaS, SaaS)
  * Differences between Cloud and On-Premise

* Successfully created an **AWS** account:
  * Activated **Free Tier**
  * Became familiar with the **AWS Management Console**
  * Understood usage limits and how to avoid unexpected costs

* Learned the main AWS service groups:
  * **Compute** (e.g., EC2)
  * **Storage** (e.g., S3)
  * **Networking** (e.g., VPC)
  * **Database** (e.g., RDS)

* Understood and used **IAM**:
  * Created and managed Users and Groups
  * Assigned permissions using Policies
  * Applied the Principle of Least Privilege
  * Recognized the importance of securing the Root account

* Successfully practiced:
  * Creating IAM Users and Groups
  * Assigning appropriate permissions based on roles
  * Verifying and testing access permissions
  * Logging in and using the system with IAM Users

* Key takeaways:
  * Do not use the Root Account for daily tasks
  * Always control access permissions to avoid security risks
  * Understand the importance of resource management from the beginning
