**Tuần 1:** [Làm quen môi trường thực tập, tạo và bảo mật tài khoản AWS, nắm vững kiến thức cơ bản về Cloud Computing và IAM, thực hành quản lý user, group và phân quyền truy cập](Week1/)

**Tuần 2:** [Triển khai và làm việc với các dịch vụ cốt lõi như EC2, S3, CloudWatch và RDS; hiểu cách cấu hình, vận hành và giám sát tài nguyên trên AWS](Week2/)

**Tuần 3:** [Mở rộng kiến thức với Route 53, Auto Scaling, AWS CLI, DynamoDB và CloudFront; bước đầu xây dựng hệ thống có khả năng mở rộng và phân phối nội dung](Week3/)

**Tuần 4:** [Tìm hiểu chuyên sâu về VPC, Subnet, Routing và các cơ chế bảo mật (Security Group, NACL); kết hợp Load Balancer và caching để tối ưu hiệu năng hệ thống](Week4/)

**Tuần 5:** [Tiếp cận mô hình serverless với Lambda và API Gateway; xây dựng API cơ bản, làm quen với Systems Manager và triển khai hạ tầng bằng CloudFormation](Week5/)

**Tuần 6:** [Ôn tập và củng cố kiến thức thông qua các bài lab thực tế; nâng cao kỹ năng triển khai EC2, VPC, Lambda, S3 và CloudFront trong hệ thống hoàn chỉnh](Week6/)

**Tuần 7:** [Làm quen với AWS CDK và tối ưu tài nguyên; thực hành giám sát hệ thống với VPC Flow Logs, quản lý chi phí và xây dựng IAM policy nâng cao](Week7/)

**Tuần 8:** [Tập trung vào bảo mật hệ thống với Security Hub, WAF, KMS, Secrets Manager và Cognito; xây dựng nền tảng bảo mật và quản lý truy cập cho ứng dụng](Week8/)

**Tuần 9:** [Tiếp cận kiến trúc event-driven với SQS và SNS; làm quen Docker, xây dựng CI/CD pipeline với CodePipeline và điều phối workflow bằng Step Functions](Week9/)

**Tuần 10:** [Bắt đầu triển khai dự án thực tế sử dụng serverless architecture; tích hợp Lambda, API Gateway, DynamoDB và SQS để xây dựng hệ thống xử lý dữ liệu](Week10/)

**Tuần 11:** [Phát triển hệ thống fullstack trên AWS với EC2; xây dựng backend bằng Python, triển khai frontend với Amplify và tích hợp authentication bằng Cognito](Week11/)

**Tuần 12:** [Hoàn thiện và triển khai dự án; tối ưu hiệu năng và chi phí, chuẩn bị tài liệu (README, proposal), tổng hợp worklog và hoàn tất báo cáo thực tập](Week12/)
